# 🎓 Student Management System

A console-based Student Management System developed using **Python** and **SQLite**. This project performs CRUD operations to manage student records.

## 🚀 Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Count Students
- Show Topper
- Calculate Average Marks
- Sort Students by Marks
- Export Data to CSV

## 🛠️ Technologies Used
- Python
- SQLite
- Git & GitHub

## ▶️ How to Run
```bash
python database.py
python main.py
```
